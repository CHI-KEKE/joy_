## 11. Salepage Collection

### 11.1 Match (dynamic support)

傳入一批 salepage ids，回傳 salepage ids 分別在哪些圈選內

<br>

```http
POST {{host}}api/salepage-collections:match
Content-Type: application/json
N1-TRACE-ID: {{traceId}}
N1-SHOP-ID: {{shopId}}

{
   "salepageIds":[
    62084
   ],
   "matchType":"FilterBySalepageCollectionIds",
   "targetSalepageCollectionIds":[
      "d_305748460756538368"
   ]
}
```
