# 商品定期購設定

## 功能概述

「定期購」是一種特殊的商品交期類型，消費者購買後可依設定的配送週期定期收到商品。

---

## 設定頁面入口

| 項目 | 說明 |
|------|------|
| **頁面名稱** | 商品管理 → 商品新增 / 商品編輯 |
| **QA 環境網址** | `https://sms.qa.91dev.tw/SalePage/Edit` |
| **設定位置** | 編輯頁「交期」區段 → 選擇「定期購」Radio Button |

---

## 前置條件

商店必須先在後台開啟定期購功能，「定期購」選項才會在商品編輯頁出現。

| 設定鍵（ShopDefault） | 說明 |
|----------------------|------|
| `IsRegularOrderEnable` | 商店層級開關，必須為 `true` |
| `RegularOrderProvider` | 定期購供應商，決定 UI 行為（`Nine1` 或 `GameBall`） |

---

## 核心資料欄位

**資料表**：`WebStoreDB.SaleProduct`

| 欄位 | 型別 | 說明 |
|------|------|------|
| `SaleProduct_ShippingTypeDef` | `int` | 交期類型，`7 = 定期購` |
| `SaleProduct_DeliveryPeriod` | `varchar` | 配送週期（以 `;` 分隔天數，例如 `7;14;30`） |
| `SaleProduct_DeliveryPeriodTypeDef` | `varchar` | 週期單位，目前固定為 `"Day"` |

**對應 Enum**：`SaleProductShippingTypeDefEnum.RegularOrder = 7`

---

## 兩種供應商的操作流程

### ✅ Nine1（91APP 自有供應商）

```
1. 進入商品編輯頁
2. 「交期」選擇「定期購」
3. 勾選配送週期 checkbox（例如：7天、14天、30天）
4. 儲存商品
```

> ⚠️ **前提**：商店需已開啟「貨到付款」付款方式，否則「定期購」選項為 disabled 狀態。

---

### ✅ GameBall（第三方供應商）

```
1. 進入商品編輯頁
2. 「交期」選擇「定期購」
3. 點擊「前往設定」連結，跳轉至第三方平台完成定期購設定
   → {RegularOrderUrl}/admin/controllers/regular/product.php?shopId={shopId}&salePageId={salePageId}
4. 返回 SMS，系統自動查詢設定狀態
   → ✅ 設定完成：顯示「定期購設定完成」
   → ❌ 尚未完成：顯示「尚未完成設定」，儲存按鈕 disabled
5. 確認設定完成後，儲存商品
```

## 資料流程

```
商品編輯頁 (productInfo.html)
  ↓ 選擇「定期購」交期 Radio Button
  ↓ Nine1：勾選配送週期 checkbox
  ↓ GameBall：前往第三方平台完成設定
  ↓ 點擊「儲存」
SalePageService.UpdateSalePageInfoWithSku()
  ↓ SaleProduct_ShippingTypeDef  = 7
  ↓ SaleProduct_DeliveryPeriod   = "7;14;30"
  ↓ SaleProduct_DeliveryPeriodTypeDef = "Day"
SCM API → 更新 WebStoreDB.SaleProduct
```

## 限制與驗證規則

| 限制 | 說明 |
|------|------|
| ❌ 不可切換交期 | 商品一旦設為定期購，交期無法再變更為其他類型 |
| ❌ 不支援組合商品 | `SalesTypeDef = "Bundle"` 的商品不顯示定期購選項 |
| ❌ 不支援虛擬商品 | `IsVirtualProduct = true` 的商品不顯示定期購選項 |
| ❌ 不可與預訂取貨併用 | Validator 中限制，已啟用預訂取貨的商品不允許設為定期購 |
| ⚠️ Nine1 需貨到付款 | Nine1 供應商需商店已開啟貨到付款，否則選項 disabled |
| ⚠️ GameBall 需完成設定 | 第三方設定未完成前，商品儲存按鈕 disabled |

---

## 商品清單顯示

商品清單頁（`/SalePage/List`）中，定期購商品於「交期」欄位顯示「定期購」標籤，方便識別。
