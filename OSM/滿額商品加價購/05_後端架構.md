# 後端架構

## 一、CartReachPriceExtraPurchaseRuleService

**路徑**：`BusinessLogic\Services\PromotionEngines\CartReachPriceExtraPurchaseRuleService.cs`  
**命名空間**：`NineYi.Sms.BL.Services.PromotionEngines`

### 說明
購物車滿額加價購活動的 Rule Service，實作 `IPromotionEngineRuleService` 介面，繼承自 `PromotionEngineRuleBaseService`。

### 主要方法

#### GetPromotionEngineRule()
```csharp
public string GetPromotionEngineRule(int promotionEngineId, PromotionEngineBaseEntity entity, ...)
```
**功能**：將前端傳入的 `ExtraPurchaseRuleList` 組成 N1PromotionService 規則 JSON。

**核心邏輯**：
```csharp
// 組成活動條件階層（滿額門檻）
var pairs = new List<CartExtraPurchaseCondition>();
entity.ExtraPurchaseRuleList?.ForEach(rule =>
{
    pairs.Add(new CartExtraPurchaseCondition
    {
        ConditionId = rule.ConditionId,
        ReachPrice = rule.ReachPrice,  // 金額門檻
        ReachPiece = 0                 // 件數門檻固定為 0（滿額模式）
    });
});
```

#### ParsePromotionEngineRuleObject()
```csharp
public PromotionEngineRuleEntity ParsePromotionEngineRuleObject(string jsonRule)
```
**功能**：將 N1PromotionService 回傳的 Rule JSON 解析成 `PromotionEngineRuleEntity`。

**解析的欄位**：
- `CartExtraPurchaseConditions`（`CartExtraPurchaseCondition` 清單）
- `CartExtraPurchaseThresholdEligibleSetting`（門檻計算設定）
- `IsCartExtraPurchaseThresholdEligible`（由 `IsCartExtraPurchaseRegisterReachPiece` + `IsCartExtraPurchaseRegisterReachPrice` + `IsCartExtraPurchaseDiscountReachPriceWithFreeGift` 三個條件 AND 計算）
