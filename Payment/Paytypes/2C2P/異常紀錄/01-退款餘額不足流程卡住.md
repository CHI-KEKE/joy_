# 異常紀錄 — 退款餘額不足導致 TwoCTwoPRefundRequestFinish 卡住

## 問題描述

執行退款時遇到 Error Code `46`（餘額不足），導致流程進入異常循環。

<br>

## 事件流程

```
退款時遇到餘額不足（Error Code: 46）
  → 狀態進入 RefundRequestProcessing
  → gersalesorder timeout
  → 接下來判斷 refund = finish 才回壓大表
  → 執行 redo → finish
  → 目前狀態已正常
```

<br>
