# 異常案例 1.6 — APP 拿到 Key 但不是這個帳戶的 Key（Stripe API）

## 現象

![alt text](../../Img/image-14.png)

## 說明

APP 成功取得 PublishableKey，但該 Key 屬於另一個 Stripe 帳戶，導致 Stripe API 驗證失敗。

## 排查方向

1. 確認 ShopDefault 中 `StripeAccountType` 設定是否正確
2. 確認 ShopSecret 中對應帳戶類型的 `PublishableKey` 是否為該商店的正確 Key
3. 注意 Custom / Standard 帳戶的 Key 不可混用
