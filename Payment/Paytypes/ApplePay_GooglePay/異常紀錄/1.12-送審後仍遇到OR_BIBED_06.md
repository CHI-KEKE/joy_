# 異常案例 1.12 — QA 正式環境 2 號店送審後仍遇到 OR_BIBED_06

## 現象

QA 正式環境 2 號店在送審完成後，發現仍遇到錯誤，錯誤碼為 **OR_BIBED_06**。

## 根本原因

Stripe PublishableKey 與 GooglePay SDK Mode（Test / Live）不一致。

## 判斷機制

MWeb 會根據 Stripe PublishableKey 開頭來決定 SDK 模式：

| PublishableKey 開頭 | SDK 模式 |
|---------------------|---------|
| `pk_test_` | Test Mode |
| `pk_live_` | Live Mode |

若 PublishableKey 有變動，則會抓取 Config 預設值 `Stripe.DefaultMode`。

## 解法

確認商店的 PublishableKey 與 GooglePay SDK 的 Mode 設定一致（Live 用 `pk_live_`，Test 用 `pk_test_`），避免模式對不上導致 OR_BIBED_06。
