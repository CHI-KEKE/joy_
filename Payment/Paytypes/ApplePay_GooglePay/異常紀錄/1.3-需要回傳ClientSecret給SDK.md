# 異常案例 1.3 — 需要回傳 ClientSecret 給 SDK

## 現象

![alt text](../../Img/image-3.png)

## 原因

如果不回傳 ClientSecret，使用者會在 ApplePay UI 看到紅色驚嘆號付款失敗，但實際上在後台是有成單的，這應該是會被投訴。

## 實現邏輯

結帳完成後，PMW 會把 ClientSecret 帶給 MWeb，並使用方法：

```
StripePayChannelService.ChangeExtendInfoAfterPaymentResult
```

附加在 `context.ThirdPartyPayment.ExtendInfo` 節點，`CompleteForNewCart` 回給前端完成付款流程。
