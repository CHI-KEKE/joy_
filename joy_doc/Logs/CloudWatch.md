## 📊 站台狀態查詢

**監控面板存取**：

<br>

透過 CloudWatch Dashboard 可以查詢各站台的即時狀態和效能指標

<br>

**查詢步驟**：

<br>

1. 登入 AWS CloudWatch 控制台
2. 導航至 Dashboards 區域
3. 選擇對應的站台監控面板
4. 查看即時監控數據和告警狀態

<br>

**監控指標**：

<br>

- CPU 使用率
- 記憶體使用情況
- 網路流量
- 應用程式回應時間
- 錯誤率統計
