# 滿額加價購 — 商品頁設定關閉（IsClosed）邏輯現狀

## 概述

說明滿額加價購（CartExtraPurchaseService）在顯示加購品清單時，
對於「商品頁設定關閉」的處理現狀，以及實作排除所需的修改方式。

---

## IsClosed 的資料來源

`IsClosed` 是 DB 直接欄位，整條路徑無任何計算或轉換：

```
DB: SalePage.SalePage_IsClosed（bit）
  ↓
SPS index: isClosed
  ↓
SPS API 回傳 JSON: { "isClosed": true/false }
  ↓
C# 反序列化: SalepageListingResponseDataSalepageModel.IsClosed（bool）
```

```csharp
// SalepageListingResponseDataSalepageModel.cs, L120-121
[JsonPropertyName("isClosed")]
public bool IsClosed { get; set; }
```

## GetSalepageListByIdsResponse 的狀態欄位現狀

| 欄位 | 型別 | 說明 | 與 IsClosed 的關係 |
|------|------|------|--------------------|
| `StatusDef` | string | 商品頁是否開賣狀態 | 只有 `"Hidden"` / `"Normal"` 兩個值，對應 IsInvisible，**不代表 IsClosed** |
| `IsClosed` | ❌ **不存在** | — | 此欄位不在 GetSalepageListByIdsResponse 中 |

### StatusDef 的實際賦值邏輯

```csharp
// SalepageService.cs（SPL 內部）
StatusDef = salepage.IsInvisible ? "Hidden" : "Normal"
```

- `"Hidden"` → 商品為隱賣（IsInvisible = true）
- `"Normal"` → 商品正常顯示
- **不包含 IsClosed 狀態**

## IsClosed 的完整資料來源追蹤

`salepage.IsClosed`（在 `SalepageListingResponseDataSalepageModel` 中）是直接對應 DB 欄位：

```
DB: SalePage.SalePage_IsClosed（bit）
  ↓
SPS index: isClosed
  ↓
SPS API 回傳 JSON: { "isClosed": true/false }
  ↓
C# 反序列化: SalepageListingResponseDataSalepageModel.IsClosed（bool）
```

整條路徑無任何計算或轉換，**直接等於 DB `SalePage_IsClosed` 欄位值**。

## 解決方案：SPL 內部補漏（工程量小）

`GetSalepageListByIdsResponse.MapResponse()` 的輸入參數也是 `SalepageListingResponseDataSalepageModel`，
`salepage.IsClosed` 值在 SPL 內部已是現成的，只是**沒有被帶出去**。

### Step 1：`GetSalepageListByIdsResponse` 新增屬性

```csharp
// GetSalepageListByIdsResponse.cs
/// <summary>
/// 此商品頁是否關閉
/// </summary>
public bool IsClosed { get; set; }
```

### Step 2：`MapResponse()` 補上映射（加一行）

```csharp
// SalepageService.cs MapResponse() 內
private List<GetSalepageListByIdsResponse> MapResponse(
    List<SalepageListingResponseDataSalepageModel> salepages)
{
    var response = salepages.Select(salepage => new GetSalepageListByIdsResponse
    {
        // ... 現有欄位 ...
        IsClosed = salepage.IsClosed,   // ← 補這一行（值來自 DB SalePage_IsClosed）
    }).ToList();
}
```

### Step 3：`CartExtraPurchaseService.BuildPromotionEntitiesAsync()` 加排除

```csharp
if (salepage.IsClosed) continue;   // 排除商品頁已關閉的商品
```

**此方案完全在 SPL Repo 內部完成，不需跨 Repo，不需新增 API 參數。**
