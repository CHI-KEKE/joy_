# 完整流程追蹤

## 場景設定

- 商品 A：設有「購買資格限制」（僅金卡會員可購買）
- 商品 A 同時是某滿額加價購活動的加購品
- 測試會員：一般會員（不符合金卡資格，但達成滿額門檻）

---

## 場景一：加購品選購清單（ExtraPurchaseProductList）

```
使用者點開加購 Modal
  └─ ExtraPurchaseProductList.tsx 渲染加購品清單
       └─ ThemeCoreProductCard
            isRestricted={false}   ← ✅ 已硬碼 false，不顯示限制標籤
```

**結論**：加購品選購清單的商品卡 **已正確處理**，不顯示購買資格限制標籤。

---

## 場景二：加購品加入購物車後的購物車頁

```
① 使用者選擇加購品加入購物車
   OptionalTypeDef = "CartReachPriceExtraPurchase"
   OptionalTypeId  = specialPriceId

② Cart API 取得購物車（GetCartProcessorLayers）
   CartCreateProcessor → GetShoppingCartRegularSalePageProcessor → ...
   ← 不含 ArrangeUnmatchPurchasesQualificationProcessor
   ← 此時商品正常出現在購物車

③ Cart API 重新計算（LogisticCalculateProcessorLayers）
   CalculatePromotionProcessor
     └─ 呼叫 Promotion Engine API
     └─ Promotion Engine：商品 A 不符合購買資格
          → Flags 加入 "RestrictedPurchasesNotMeet"
     └─ salePage.PromotionFlags = { "RestrictedPurchasesNotMeet", ... }
   
   ⚠️ ArrangeUnmatchPurchasesQualificationProcessor
     └─ 找到 PromotionFlags 含 RestrictedPurchasesNotMeet 的商品
     └─ 把商品 A（加購品）移入 UnmatchPurchasesQualificationSalePageList
     └─ 主清單 SalepageGroupList 移除商品 A

④ Cart API 回傳給 MWeb
   UnmatchPurchasesQualificationSalePageList：[商品 A]
   SalepageGroupList：[不含商品 A]

⑤ MWeb 前端 productList/index.tsx
   isUnmatchPurchaseQualification: true → 商品 A 呈現為「購買資格限制」區塊

⑥ useProductCard.tsx
   if (isUnmatchPurchaseQualification) {
       setIsDisabled(true);
       setMaskType(MaskType.Unpurchasable);  ← 灰底 + 資格限制標籤
   }
```

**問題**：商品 A 在購物車中顯示「購買資格限制」標籤且無法操作。

---

## 場景三：結帳流程

```
使用者點擊結帳
  └─ CheckoutCreateProcessorLayers
       └─ MergeRequestAndCartContextProcessor
            if (SalepageGroupList 沒有商品  ← 商品 A 已被移出
               AND UnmatchPurchasesQualificationSalePageList 有商品)
            {
                throw CheckoutCreateException(
                    UnmatchPurchaseQualificationSalePageOnly)  ← ⚠️ 阻擋結帳
            }
```

**問題**：若購物車只有商品 A（加購品），結帳被完全阻擋。

## 各層狀態總覽

| 層 | 現行行為 | 期望行為 |
|----|---------|---------|
| ExtraPurchaseProductList | ✅ `isRestricted={false}` 已正確 | 不變 |
| ArrangeUnmatchPurchasesQualification | ❌ 把加購品也移出主清單 | 加購品應跳過 |
| MWeb 購物車顯示 | ❌ 顯示限制標籤 | 自動修正（上游修好後） |
| MergeRequestAndCartContextProcessor | ❌ 因空購物車拋 exception | 自動修正（上游修好後） |
