# 03 — 完整流程追蹤（分享活動）

## 舊流程：一般商品「分享活動」加入購物車

```
商品 A（IsShareToBuy = true）
    │
    ▼
[商品頁 WebAPI] SalePageController
    • result.IsShareToBuy = true
    • 回傳給前端
    │
    ▼
[MWeb AngularJS 前端] 商品頁
    • 接收 IsShareToBuy = true
    • 傳入 addToCartDirective: nsIsShared = false（尚未分享）
    │
    ▼
[MWeb AngularJS] addToCartDirectiveController.ts
    • this.$scope.IsShared = false
    • CurrentAddToCartType = AddToCartType.ShareToBuy  ← 觸發分享流程
    │
    ▼
[MWeb 前端] FB 分享 Dialog 彈出
    • 使用者必須分享至 Facebook
    • 分享完成 → IsShared = true → 可加入購物車
    • 未分享 → 無法加入購物車 ❌（一般商品的正常業務邏輯）
```

## 新流程：滿額加購品（已確認，現狀正確）

```
消費者在 Web 端達成滿額門檻
    │
    ▼
[MWeb CartClientApp] cartReachPriceExtraPurchaseBlock.tsx
    • 顯示加購選單入口
    │
    ▼
[SPL] CartExtraPurchaseService.BuildSalePageDictionaryAsync()
    • 呼叫 SalepageService.SalepagesSearchByIdsAsync()
    │
    ▼
[SPL] SalepageService.SalepagesSearchByIdsAsync() L235
    • requestEntity.Filters.IsIncludingShareToBuy = true  ← 硬設包含
    • → SPS 不過濾 IsShareToBuy = true 的商品
    • 商品 A 正常回傳 ✅
    │
    ▼
[MWeb CartClientApp] 加購 Modal
    • ISalepage / ICartExtraPurchaseSalePage 沒有 isShareToBuy 欄位
    • 加購 Modal 正常顯示商品 A ✅
    • 不顯示任何分享提示 ✅
    │
    ▼
[MWeb CartClientApp] CartExtraPurchaseProvider.tsx
    • 消費者選擇商品 A → 確認加購
    • 直接呼叫 Cart Domain API
    • ← 完全不走 addToCartDirective ✅
    • ← 不觸發 FB 分享 Dialog ✅
    │
    ▼
[Cart Domain] 加入購物車 / 所有 Processor
    • SalepageEntity 無 IsShareToBuy 欄位
    • 不檢查、不阻擋 ✅
    │
    ▼
[Cart Domain] 結帳 Checkout Processors
    • 完全無 IsShareToBuy 相關判斷
    • 加購品可正常結帳 ✅
    │
    ▼
[MWeb BLV2] 購物車計算 Processors
    • 完全無 IsShareToBuy 相關判斷 ✅
    │
    ▼
[MWeb CartClientApp 購物車] ISalepage
    • 無 isShareToBuy 欄位
    • 購物車不顯示分享活動標籤 ✅
```

## 新舊流程對比

| 步驟 | 舊流程（一般商品加購） | 新流程（滿額加購品） |
|------|---------------------|-------------------|
| 進入點 | 商品頁「加入購物車」| 購物車「加購選單」|
| SPL filter | SPS 預設排除 ShareToBuy | `IsIncludingShareToBuy = true` 強制包含 |
| 阻斷機制 | `addToCartDirective.IsShared` 判斷 | 不存在（不走此 Directive）|
| 分享 Dialog | ✅ 觸發（若 IsShared=false）| ❌ 不觸發 ✅ |
| Cart Domain | 不讀取 IsShareToBuy | 不讀取 IsShareToBuy ✅ |
| 結帳驗證 | 無阻擋 | 無阻擋 ✅ |

## 關鍵檔案位置

| 檔案 | 關鍵行 | 說明 |
|------|-------|------|
| `SalepageService.cs` | L235 `IsIncludingShareToBuy = true` | **決定性設定**：加購清單包含分享活動商品 |
| `CartExtraPurchaseService.cs` | `BuildSalePageDictionaryAsync()` | 呼叫上述方法的入口 |
| `addToCartDirectiveController.ts` | `IsShared` 判斷 | 舊流程分享阻斷點（加購品不走此路徑）|
| `CartExtraPurchaseProvider.tsx` | 加購加入購物車 | 新流程：不走 addToCartDirective |
