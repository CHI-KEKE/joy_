# 02 — 分享活動機制

> 本文件只討論**商品頁維度**：`SalePage.IsShareToBuy`。

---

## 資料層

### DB 欄位

```
SalePage.IsShareToBuy (bool)
SalePage.SalePage_IsShareToBuy (DB column)
```

### MWeb 讀取路徑（商品頁 API）

```csharp
// WebStore/DA/WebStoreDB/SalePageDAHelper.cs
GetSalePageIsShareToBuy(salePageId) → bool

// WebStore/Frontend/BL/SalePage/SalePageManager.cs
GetSalePageIsShareToBuy(salePageId, traceLog) → bool
```

`IsShareToBuy` 透過商品頁 API（`/WebAPI/SalePage`）回傳給前端，用於商品頁加入購物車前的分享驗證。

---

## 各層對 `IsShareToBuy` 的讀取狀況

### ✅ 有讀取（影響商品頁加購行為）

| 層級 | 位置 | 說明 |
|------|------|------|
| WebAPI SalePageController | L396 `result.IsShareToBuy` | 商品頁 API 回傳，僅用於商品頁 |
| MWeb AngularJS addToCartDirectiveController | `CurrentAddToCartType = AddToCartType.ShareToBuy` | **舊版分享阻斷點**，只在商品頁加購觸發 |

### ❌ 不讀取（加購品流程）

| 層級 | 位置 | 說明 |
|------|------|------|
| SPL `CartExtraPurchaseService` | `BuildSalePageDictionaryAsync()` | 呼叫 `SalepagesSearchByIdsAsync()`（已設 `IsIncludingShareToBuy = true`） |
| SPL `SalepageService.SalepagesSearchByIdsAsync()` | L235 | 硬設 `IsIncludingShareToBuy = true`，ShareToBuy 商品直接包含在結果中 |
| Cart Domain `SalepageEntity` | 所有 Processor | 無 `IsShareToBuy` 欄位 |
| CartClientApp `ISalepage` 介面 | types.ts | 無 `isShareToBuy` 欄位，不顯示分享提示 |
| Cart Domain 結帳 Processor | BL.Services\Processor\Checkout | 完全無此欄位的讀取 |
| MWeb BLV2 ShoppingCartsV2 | 所有 Processor | 完全無 `IsShareToBuy` 判斷 |

## SPL 的 `IsIncludingShareToBuy` 機制

SPS（Salepage Presentation Service）預設會**排除** `IsShareToBuy = true` 的商品。  
若要包含這類商品，需明確設定 `IsIncludingShareToBuy = true`。

```csharp
// SalepageSearchPresentationFilters.cs L69
[JsonPropertyName("isIncludingShareToBuy")]
public bool? IsIncludingShareToBuy { get; set; }
```

### 各流程對此 flag 的設定

| 流程 | 設定值 | 結果 |
|------|--------|------|
| `SalepageService.SalepagesSearchByIdsAsync()` L235 | `= true` | ShareToBuy 商品包含在結果中 ✅ |
| `SalepageExtraPurchaseService.cs` L354（舊版 AddOns） | `= true` | ShareToBuy 商品包含在結果中 ✅ |
| SPS 預設（未設定時） | `null` / `false` | ShareToBuy 商品被排除 |

由於 `CartExtraPurchaseService` 呼叫 `SalepageService.SalepagesSearchByIdsAsync()`，  
自動繼承 `IsIncludingShareToBuy = true` 的設定，**分享活動商品可正常出現在加購清單**。

## 舊版 AngularJS 流程（僅供參考）

舊版 `addToCartDirectiveController.ts` 的分享阻斷邏輯：

```typescript
// 依照 IsShared 狀態決定加入購物車行為
CurrentAddToCartType = this.$scope.IsShared
    ? AddToCartType.Normal       // 已分享 → 正常加購
    : AddToCartType.ShareToBuy;  // 未分享 → 觸發 FB 分享 Dialog
```

此邏輯只在商品頁的「加入購物車」按鈕路徑觸發，  
新版加購流程（`CartExtraPurchaseProvider.tsx`）完全不走此路徑。
