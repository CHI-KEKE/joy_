## 改動內容

### 修改前

```csharp
private string GetShippingTypeDef(SalePageEntityForCart salePageEntityForCart)
{
    var shippingTypeDef = ((SaleProductShippingTypeEnum)salePageEntityForCart.SaleProductShippingTypeDef).ToString();

    //// 若是商品加價購的加購品 & 交期 = 定期購，將交期換成一般避免加購品被定期購買
    if (salePageEntityForCart.IsSubAddOns() && shippingTypeDef == nameof(SaleProductShippingTypeEnum.RegularOrder))
    {
        shippingTypeDef = nameof(SaleProductShippingTypeEnum.Normal);
    }

    return shippingTypeDef;
}
```

### 修改後

```csharp
private string GetShippingTypeDef(SalePageEntityForCart salePageEntityForCart)
{
    var shippingTypeDef = ((SaleProductShippingTypeEnum)salePageEntityForCart.SaleProductShippingTypeDef).ToString();

    //// 若是商品加價購的加購品 & 交期 = 定期購，將交期換成一般避免加購品被定期購買
    if (salePageEntityForCart.IsSubAddOns() && shippingTypeDef == nameof(SaleProductShippingTypeEnum.RegularOrder))
    {
        shippingTypeDef = nameof(SaleProductShippingTypeEnum.Normal);
    }

    //// 若是購物車滿額/滿件加價購活動的加購品 & 交期 = 定期購，同樣降級為一般交期
    //// 避免加購品以定期購方式被購買（滿額加購活動不支援定期購邏輯）
    if (salePageEntityForCart.IsCartExtraPurchase && shippingTypeDef == nameof(SaleProductShippingTypeEnum.RegularOrder))
    {
        shippingTypeDef = nameof(SaleProductShippingTypeEnum.Normal);
    }

    return shippingTypeDef;
}
```

## 為什麼只改這一處

### 為什麼不改 Shopping API？

Shopping API 的 `CartCreateProcessor` 直接 pass-through Cart API 的 `CartEntity`：

```csharp
// Shopping CartCreateProcessor.cs
context.Data = result.Data!;
```

`SaleProductShippingTypeDef` 的值完全由 Cart API 決定，Shopping API 不做額外處理。

### 為什麼不需要調整 Processor 順序？

`GetShippingTypeDef()` 在 **[1] CartCreateProcessor** 執行，`SaleProductShippingTypeDef` 設為 `"Normal"` 後，**[3] `GetShoppingCartRegularSalePageProcessor`** 的篩選條件（`== "RegularOrder"`）自然不成立，不進入定期購分支。Processor 順序本身沒有問題。

### 為什麼不改 MWeb？

MWeb 已正確傳送 `optionalTypeDef = 'CartReachPriceExtraPurchase'` 和 `optionalTypeId = specialPriceId`，實作已完成。

## 預期效果

| 場景 | `IsCartExtraPurchase` | `GetShippingTypeDef()` 結果 | 前端標籤 |
|------|----------------------|--------------------------|---------|
| 商品透過商品頁加入 | `false` | `"RegularOrder"` | 「定期購」✅ |

## 影響範圍評估

| 範圍 | 影響 |
|------|------|
| 一般商品（非定期購） | ✅ 不受影響（`shippingTypeDef != "RegularOrder"`，if 不成立） |
| 定期購商品透過商品頁加入 | ✅ 不受影響（`IsCartExtraPurchase == false`，if 不成立） |
| 商品加價購子品（IsSubAddOns） | ✅ 不受影響（已由上方 if 處理） |
| **滿額加價購加購品（定期購商品）** | ✅ 正確降級為 Normal |
| 滿件加價購加購品（定期購商品） | ✅ 同樣正確降級（`IsCartExtraPurchase` 也涵蓋 `CartReachPieceExtraPurchase`） |
