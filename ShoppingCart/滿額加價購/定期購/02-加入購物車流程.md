## 關鍵函式說明

### `getSpecifyOptionalTypeDef`

```typescript
// CartExtraPurchaseProvider.tsx
const validTypes: Record<string, SalePageOptionalType> = {
    CartReachPriceExtraPurchase: 'CartReachPriceExtraPurchase',
    CartReachPieceExtraPurchase: 'CartReachPieceExtraPurchase',
};

const getSpecifyOptionalTypeDef = (typeDef?: string | null): SalePageOptionalType | undefined => {
    return typeDef ? validTypes[typeDef] : undefined;
};
```

`promotionData.typeDef` 來自 API 回傳的促銷活動類型字串，與 `validTypes` 對照後得到 `'CartReachPriceExtraPurchase'`。

### `optionalTypeId = sku.specialPriceId`

`specialPriceId` 是此加購條件（condition）下這個 SKU 的特價設定 ID，必須 `> 0` 才能讓 `IsCartExtraPurchase` 判斷為 `true`。
