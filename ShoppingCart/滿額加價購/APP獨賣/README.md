> ⚠️ 本文件只討論**商品頁維度**（`SalePage_IsAPPOnly`），不討論 PE 活動通路限制。

- **現狀**：商品頁的 `SalePage_IsAPPOnly = true` 在整個新版加購品流程中**完全未被讀取**

- **不需要改 Code**：現狀已符合 Story 需求（加購品確實繞過了一般商品的 APP 獨賣設定）
