# 搶先曝光（IsComingSoon）機制說明

## 定義

搶先曝光是一種商品上架模式，讓商品在正式開賣前就能在列表中曝光，但不允許購買。

## 判斷邏輯

```csharp
// GetSalepageListByIdsResponse.cs（SPL）
public bool IsComingSoon =>
    ListingStartDateTime <= DateTime.Now &&   // 已到上架時間
    DateTime.Now < SellingStartDateTime &&    // 尚未到開賣時間
    EnableIsComingSoon;                       // 商店開啟搶先曝光功能
```

三個條件必須同時成立：

| 條件 | 說明 |
|------|------|
| `ListingStartDateTime <= now` | 商品已可「展示」 |
| `now < SellingStartDateTime` | 商品尚未可「販售」 |
| `EnableIsComingSoon == true` | 商店層級開關（`ShopDefault Key=86`）為開啟 |

## 時間軸示意

```
─────────────────────────────────────────────────────────────►  時間

        ListingStartDateTime            SellingStartDateTime
               │                               │
               ▼                               ▼
───────────────●───────────────────────────────●──────────────
               │◄──── 搶先曝光區間（IsComingSoon=true）────►│
               │                               │
          商品出現在列表                    商品可購買
```

## EnableIsComingSoon 開關

- **位置**：`ShopDefault` 表，Key = 86
- **預設值**：`true`（商店預設開啟搶先曝光）
- **作用範圍**：控制整個商店的搶先曝光顯示行為

## 與加購清單的關係

- **一般商品列表**：會顯示搶先曝光商品（提前曝光），但按鈕 disabled
- **加購清單**：**不應顯示**搶先曝光商品（Story #596792 AC）
- **判斷依據**：AC 明確指定用 `SellingStartDateTime`，且**忽略 `EnableIsComingSoon`**

## 重要區別

```
IsComingSoon（現有屬性）：含 EnableIsComingSoon 商店開關
SellingStartDateTime > now（AC 要求）：純時間判斷，不含商店開關
```

兩者邏輯不同，加購清單應使用後者（純 `SellingStartDateTime` 判斷）。
