# 01 — 流程總覽

## 整體稽核流程

```
[入口] CartExtraPurchaseAuditProcessor.ExecuteAsync(context)
            │
            ▼
[Step 0] IsSkip 判斷
  - context.IsForCache == false → 跳過（non-cache 路徑不執行）
  - 沒有任何 IsCartExtraPurchase == true 的 SalePage → 跳過
            │
            ▼
[Step 1] 取得訂單時間
  var orderDateTime = context.ShoppingCartV2.ShoppingDateTime
                      ?? context.EventOrderDateTime;
            │
            ▼
[Step 2] 取得下單當下在期的滿額加價購活動 IDs
  POST /api/promotion-rule/ongoing-list
  {
    Type:      [CartReachPriceExtraPurchase],
    StartDate: orderDateTime,
    EndDate:   DateTime.Now
  }
  → List<{ Id (PromotionEngineId), TypeDef, ... }>

  語意：活動在 orderDateTime 之後才結束，且稽核當下已開始
            │
            ▼
[Step 3] 收集所有加購品的 SalePageId
  var extraPurchaseSalePageIds =
    context.ShoppingCartV2.SalePageGroupList
      .SelectMany(g => g.SalePageList)
      .Where(s => s.IsCartExtraPurchase == true)
      .Select(s => s.SalePageId)
      .Distinct()
      .ToList();
            │
            ▼
[Step 4] 對每個 PromotionEngineId 查詢活動設定的加購價
  foreach promotionEngineId in ongoingPromotionIds:
    POST /api/promotion-price-lists/get-promotion-price
    {
      PromotionEngineId: promotionEngineId,
      SalePageIds:       extraPurchaseSalePageIds
    }
    → List<{ Id(SpecialPriceId), SalepageId, SaleProductSKUId, TypeValue(加購價), ConditionId }>

  彙總所有結果：allSpecialPrices (去重)
            │
            ▼
[Step 5] 比對每筆加購品
  foreach extraPurchaseSalePage:
    matchKey = (SalePageId, SaleProductSKUId, SpecialPriceId)
    找出 allSpecialPrices 中對應那筆

    ├─ 找不到對應 → 稽核異常（Log Fail）
    │    原因：活動不存在 or SpecialPriceId 對不上
    │
    └─ 找到後比對價格
         salePage.Price == specialPrice.TypeValue ?
         ├─ 相符 → Pass
         └─ 不符 → 稽核異常（Log Fail）
            │
            ▼
[輸出] MetricsLogEntity → LogInformation("{@MetricsLog}", ...) → Slack 告警
```

---

## 關鍵欄位對照

| 用途 | Cache 欄位 | API 回傳欄位 |
|---|---|---|
| 識別加購品 | `SalePage.IsCartExtraPurchase == true` | — |
| Match key 1 | `SalePage.SalePageId` | `result.SalepageId` |
| Match key 2 | `SalePage.SaleProductSKUId` | `result.SaleProductSKUId` |
| Match key 3 | `SalePage.SpecialPriceId` | `result.Id` |
| 稽核比對 | `SalePage.Price` | `result.TypeValue` |

---

## IsSkip 條件

```csharp
public bool IsSkip(PayProcessContextEntity context)
{
    // non-cache 路徑不執行
    if (context.IsForCache == false) return true;

    // 沒有加購品則跳過
    var hasExtraPurchase = context.ShoppingCartV2.SalePageGroupList
        .SelectMany(g => g.SalePageList)
        .Any(s => s.IsCartExtraPurchase == true);

    return hasExtraPurchase == false;
}
```

## 容錯考量

| 情況 | 處理方式 |
|---|---|
| ongoing-list 回傳空清單 | Log Warning，不 Fail（活動可能已不存在） |
| get-promotion-price 某個 promotionId 回傳空 | 忽略，繼續下一個 promotionId |
| Cache 內加購品找不到對應的 specialPrice | Log Fail |
| 價格不符 | Log Fail |
