# 03 稽核邏輯設計

## IsSkip 條件

購物車中沒有任何滿額加購品時跳過：

```csharp
var hasAny = context.ShoppingCartV2.SalePageGroupList
    .SelectMany(x => x.SalePageList)
    .Any(x => x.IsCartExtraPurchase == true);

return !hasAny;
```

---

## 加購品價格稽核

> **不需查 MongoDB**，活動設定價 `NewPrice` 已存在 Cache 裡。

```
Step 1: 找出所有 IsCartExtraPurchase == true 的 SalePage
Step 2: 取其 SpecialPriceId
Step 3: 從 PromotionInfoList 找 PromotionConditionTypeDef == "CartReachPriceExtraPurchase"
        → 在 CartExtraPurchaseList 裡找 SpecialPriceId 對應的那筆
Step 4: 比對 salePage.Price == cartExtraPurchaseItem.NewPrice
        不符 → 稽核失敗（LogError + MetricsLog Fail）
```

```csharp
this._logger.LogInformation("IsForCache=false，尚未實作 DB 路徑稽核");
```
