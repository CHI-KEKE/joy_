## couponInfo.IsSelectedBeforeCheckout

```csharp
//// 由於部分配送方式P2頁才會選定地區(ex.重量計價)，故進P2前先記錄當前券選取狀態，以便後續選取適用的地區時可以還原選取狀態
foreach (var couponInfo in checkoutEntity.CouponInfoList)
{
    couponInfo.IsSelectedBeforeCheckout = couponInfo.IsSelected == true;
}

```
