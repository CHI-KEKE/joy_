# 重大 (SalpageGroup => CheckoutType.DeliveryTypeList)

```csharp
var salePageGroupDeliveryTypeList = salePageGroupList.Select(i => GetSalePageGroupDeliveryType(i.SalepageList)).ToList();
salePageGroupDeliveryTypeList => shoppingCartDeliveryTypeList
context.CheckoutType.DeliveryTypeList = shoppingCartDeliveryTypeList.ToList();

```



計算無金物流交集商品，放回可結帳清單


```csharp
//// 當無交集時，清除選擇的配送地區
if (context.Data.SalepageGroupList.Any() == false)
{
    var tempSelectedShippingArea = context.Data.SelectedShippingArea;
    context.Data.SelectedShippingArea = null;
    if (context.Data.IsGiftCouponUnMapping == false)
    {
        context.Data.SelectedShippingArea = tempSelectedShippingArea;
    }
}
```


## ArrangeCheckoutSalepage 又算金物流交集


```csharp
await _payShippingMappingService.CalculatePayDeliveryMapping(context);

var salePageGroupList = context.SalepageGroupList;
//// 取得溫層內商品頁交集結果
var salePageGroupDeliveryTypeList = salePageGroupList.Select(i => GetSalePageGroupDeliveryType(i.SalepageList)).ToList();
salePageGroupDeliveryTypeList => shoppingCartDeliveryTypeList
context.CheckoutType.DeliveryTypeList = shoppingCartDeliveryTypeList.ToList();
```
