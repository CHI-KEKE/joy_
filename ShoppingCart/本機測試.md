## 重新建置

<br>

```bash
dotnet restore
dotnet build
dotnet clean
```


## launchsettings.json

#### port

- Shopping：5566
- Cart：5567

<br>

## Settings.json

往外打的 API 要設定連接埠 5567

<br>

## 環境

settings.json 需要設定 HK QA 相關參數

<br>

## Visual Studio 設定

在 VS 上要去調整 HK-Dev 設定，才會去連 HK 資料庫

<br>

## Timeout 設定

Timeout 不能設定太短，避免連線逾時問題

<br>

## Header 設定

shopId = 2 要設定在 header 中
