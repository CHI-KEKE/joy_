## $connect Mapping Template

Route: `$connect`，使用 HTTP Integration，透過 Mapping Template 將 API Gateway 事件轉換成後端格式。

```bash
#set($connectionId = $context.connectionId)
#set($shopId = $input.params('shopId'))
#set($liveSessionId = $input.params('liveSessionId'))
#set($platformUserId = $input.params('platformUserId'))
#set($userName = $input.params('userName'))

{
  "connectionId": "$connectionId",
  "shopId": #if($shopId != "") $shopId #else 0 #end,
  "liveSessionId": #if($liveSessionId != "") "$liveSessionId" #else "" #end,
  "platformUserId": #if($platformUserId != "") "$platformUserId" #else "" #end,
  "userName": #if($userName != "") "$userName" #else "" #end
}
```

> Connection 需存的欄位：`shopId`、`liveSessionId`、`connectionId`、`直播間 role`、`platformUserId`

## $disconnect 行為

前端**不需要**主動呼叫 `DELETE /api/websocket/connections`，流程如下：

1. 使用者關閉瀏覽器分頁
2. API Gateway 偵測到連線中斷
3. 自動觸發 `$disconnect` 路由
4. API Gateway 發送 `DELETE /api/websocket/connections` 到 Backend
5. Backend 執行 `HandleDisconnectionAsync` 清理 Redis 快取

## 測試端點（Controller）

```csharp
/// <summary>
/// 測試廣播端點
/// </summary>
/// <param name="liveSessionId">直播間 ID</param>
/// <param name="message">要發送的訊息內容</param>
[HttpPost("test")]
public async Task<IActionResult> Test(string liveSessionId, string message)
{
    await this._liveSessionConnectionService.SendToWebSocketClientsAsync(liveSessionId, message);
    return Ok(new { message = "連線建立成功" });
}
```

## wscat 連線測試

安裝：`npm install -g wscat`

```bash
wscat -c "wss://trm24yg5re.execute-api.ap-southeast-1.amazonaws.com/91applive?shopId=343&liveSessionId=LS260108L00005&platformUserId=bill"
```

