# 訂閱功能問題整理

## 問題核心

訂閱相關的核心問題是 **FB Webhook `subscribed_apps` 的呼叫時機點與權限驗證不完整**。

主要體現在兩個面向：
1. 綁定粉專後未主動確認 Webhook 是否訂閱成功，導致留言事件無法推送
2. 訂閱時若粉專缺少必要權限，系統只回傳錯誤但不告知使用者缺少哪個權限
