## _purchasedItems flag

從購買商品中，撈出「應該被排除的商品」，只要命中「排除條件 Flag」，就視為 不參與計算

```csharp
/// <summary>
/// 取得排除商品清單
/// </summary>
public IEnumerable<PurchasedItem> GetExcludePurchasedItems =>
    this._purchasedItems.Where(i => i.Flags.Intersect(this._thresholdNotMeetFlags).Any() == true);
```


## 排除的商品要先扣除 TotalPrice

門檻計算只看「真的有資格參與的金額」，被排除的商品不應該影響活動成立與否

`GetExcludePurchasedItems`：把每一個排除的商品的 `SalePrice` 去移除 `TotalPrice`
