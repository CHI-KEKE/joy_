## Logic

- ShippingType = Home / LocationPickup / Oversea
- 配送方式表 : ShopShippingType
- ShippingProfile 表 : DigitalDelivery / Home / LocationPickup / Oversea
- 國家地區配送 ShippingAreaId = 0
- Shop_IsPayShippingDefault
  - shippingTypeItem.IsDefault
  - shippingTypeItem.IsSalesOrderFeeDefault
- 插入 ShopShippingType
- 測試存入
  - ShopShippingType_ShippingTypeDef - Oversea
  - ShopShippingType_ShippingProfileTypeDef - Oversea
- 編輯 shippingType 看起來也是同一支

## 語法

```sql
use ERPDB


select ShippingArea_CountryProfileId,ShippingArea_AliasCode,ShippingArea_CountryProfileId,*
from ShippingArea(nolock)
where ShippingArea_ValidFlag = 1


select CountryProfile_AliasCode,*
from CountryProfile(nolock)
where CountryProfile_Id = 30
```

shippingArea 的 countryPropfileId 會被 CountryProfile_Id 相同的 countryAliasCode 賦值
