


它是 以「州（State / City）」底下的『區、城鎮、地區、特殊地點』為單位 的資料表

AREA = 使用者地址 / 門市 / 配送 / 區域選單中，實際可以被選擇的一個「地理區域節點」


一個 Row = 一個「在某個州底下，可被系統識別與使用的地區名稱」

例如這一筆：
```bash
Area_Id = 7
Area_CityId = 1 ## 「州 / 行政區上層單位」
Area_Title = Batu Pahat
Area_LanguageKey = my_johor_batupahat
```

「馬來西亞 → Johor 州 → Batu Pahat 這個地區」
